# Vocabulary Web Scraping System

A Python project that extracts vocabulary data (meanings, pronunciation, images, and audio) from websites and stores it in structured formats (JSON, CSV).

## Tech Stack
- Python
- BeautifulSoup
- Requests

## Features
- Web scraping of vocabulary data
- Download images and audio files
- Store data in JSON and CSV formats
- Handle missing and inconsistent data

## How to Run
```bash
pip install -r requirements.txt
python project.py
